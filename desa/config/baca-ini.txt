*******************************
Folder: deskel/config
*******************************

Folder ini tempat menyimpan file konfigurasi aplikasi dan database yang dipergunakan deskel.

Konfigurasi applikasi di folder ini akan menjadi tambahan konfigurasi asli SIDeGa di
file sidega-app/config/config.php.

Konfigurasi database di folder ini akan menggantikan konfigurasi asli SIDeGa di
file sidega-app/config/database.php.
